# Portfolio Project

A full-stack portfolio application with a React client and an Express server.

## Features
- Desktop-style portfolio UI
- Project management backend
- Contact form support
- Admin authentication

## Project Structure
- client/: React frontend
- server/: Express backend

## Setup
1. Install dependencies for both client and server.
2. Create a .env file in the project root with your environment variables.
3. Start the backend and frontend.

## Quick Start
```bash
cd client
npm install
npm run dev
```

```bash
cd server
npm install
node index.js
```

## Notes
Update the environment variables in .env before running the app.
